# Delete Empty Stickies

A lightweight Miro Web SDK app that deletes empty or whitespace-only sticky notes from your current board selection.

## Usage

1. Select one or more sticky notes on your Miro board
2. Click the app icon in the toolbar
3. Any sticky notes with empty or whitespace-only content will be deleted

## Development

Install dependencies:
```bash
npm install
```

Start the local dev server:
```bash
npm start
```

The app runs at `http://localhost:3000`. Set this as your app URL in the [Miro Developer Hub](https://developers.miro.com).

## Requirements

- Node.js >= 20.9.0
- A Miro account with an app configured in the Developer Hub
